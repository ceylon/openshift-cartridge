ceylon help

Displays help information about other Ceylon tools

IndexSynopsisDescriptionOptions

Synopsis
ceylon help [--include-hidden] [--] [<tool>]

Description

If a <tool> is given, displays help about that ceylon tool on the standard output.

If no <tool> is given, displays the synopsis of the top level ceylon command.

Options
--include-hidden

This option is not documented.
