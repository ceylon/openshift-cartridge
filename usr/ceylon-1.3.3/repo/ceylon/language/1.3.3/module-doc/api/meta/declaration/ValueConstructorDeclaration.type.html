ceylon.language.meta.declaration
interfaceValueConstructorDeclaration
satisfies GettableDeclaration & ConstructorDeclaration
OverviewPackageAttributesMethods
  • Documentation
  • Type Hierarchy
  • Supertype Hierarchy

Declaration model for value constructors, for example

class Currency {
    "The US Dollar"
    shared new usd {}
    // ...
}

ValueConstructorDeclaration dollars = `new Currency.usd`;
See also CallableConstructorDeclaration
Since 1.2.0
